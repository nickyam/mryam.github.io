多种方法，帮你剪藏网页文章到 obsidian 中

发表于 2023-06-25 分类于 软件资讯 本文字数： 776 阅读时长 ≈ 3 分钟

剪藏到 OB，就是将网络上值得收藏的文章放在 obsidian 的笔记库中，等待后续回看（稍后读），以及与自己过往思维进行链接。除此以外，还能避免类似网站关停、文章删除、图片失效等问题……

剪藏文章到 Obsidian 有多种方式，我将从简单到复杂一级一级介绍。

1 复制到 Obsidian 编辑器

通过直接复制网页内容，在 obsidian 的编辑器中将自动转化为 markdown 格式。

最新的版本更新 1.3.3 还强化了复制表格的能力，使得 ob 在复制网页上的简单表格时，自动转化为 markdown 语法的表格。

1.1 图片特别处理

直接复制的图片实际是该图片的 URL，只能在有网络时查看。

如果想将复制的图片存储到本地，配合 obsidian 插件 「local image plus」，可以将图片一键下载到本地的。

打开对应选项后，将网页的图片链接粘贴在 Obsidian 的编辑器中后，更是会自动把图片下载到设置好的附件文件夹中。

如何设置附件文件夹请看：2a Obsidian 文件列表：知识管理的基石（2）

1.2 复杂表格

如果想在 obsidian 中粘贴以下这种带合并单元格的复杂表格。

第 1 步，复制网页的复杂表格。

第 2 步，安装 Advanced Paste 插件，记住一定要重启 obsidian，不然粘贴东西会出现重复粘贴两次的问题。

第 3 步，命令面板中调用 Advanced Paste:Raw HTML 命令，就能粘贴复杂表格了。

2 借助浏览器插件

通过一些浏览器插件，我们可以将直接网页转换为 Markdown 格式。

这里我只介绍 maoxian web clipper

maoxian web clipper 软件介绍可以看：MaoXian Web Clipper - 将网页剪藏保存到本地硬盘的浏览器网摘扩展插件 - 异次元软件下载

该浏览器插件的官方地址：https://github.com/mika-cn/maoxian-web-clipper

需要注意的是：因为该插件支持剪藏 html、markdown 两种格式，所以存储设置需要手动选择「markdown」

通过设置剪藏的模版，为剪藏的文件增加 YAML 区域，对文件添加元数据，自说明自已的来源，可更方便 Obsidian 管理和检索，

3 借助第三方应用

将目光脱离 obsidian， 我们可以利用专业的第三方应用实现剪藏的目的，如 Cubox、EverNote、Omnivore 等等。

比如 Omnivore，它支持：

  • 稍后读功能：支持保存网页链接、Newsletter 订阅、文档等内容。

  • 支持高亮、批注。

  • 支持标签、过滤、高级搜索等功能；

  • 支持全平台使用。

  • 支持与 Obsidian、Logseq、Readwise 的内容集成

Omnivore 官方提供了自己开发的 Obsidian 插件 obsidian-omnivore，可以直接将 Omnivore 中的剪藏同步到 Obsidian 中。

更多信息可去试用 Omnivore，这里不过多介绍。

4 总结

以上介绍了多种剪藏文章的方式，希望你能找到适合自已的工具。
